# Tetris Ai with Python

An AI implementation to play Tetris

## Tools used

    IDE: Pycharm Community - 2021 1.2
    Language: Python 3.9

## Installation

You will need the following packages installed to get this project running:
* pygame 
* numpy


    pip install pygame
    pip install numpy
